整数类型：

int 是有符号整数类型，根据不同的操作系统和架构，其位数可能是 32 位或 64 位。在大多数情况下，它与 int32 或 int64 具有相同的位数和表示范围。
int8 是有符号的 8 位整数类型，表示范围是 -128 到 127。
int16 是有符号的 16 位整数类型，表示范围是 -32768 到 32767。
int32 是有符号的 32 位整数类型，表示范围是 -2147483648 到 2147483647。
int64 是有符号的 64 位整数类型，表示范围是 -9223372036854775808 到 9223372036854775807。
uint 是无符号整数类型，根据不同的操作系统和架构，其位数可能是 32 位或 64 位。在大多数情况下，它与 uint32 或 uint64 具有相同的位数和表示范围。
uint8 是无符号的 8 位整数类型，表示范围是 0 到 255。
uint16 是无符号的 16 位整数类型，表示范围是 0 到 65535。
uint32 是无符号的 32 位整数类型，表示范围是 0 到 4294967295。
uint64 是无符号的 64 位整数类型，表示范围是 0 到 18446744073709551615。
浮点数类型：

float32 是单精度浮点数类型，表示范围是约 -3.4e38 到 3.4e38，可以精确到大约 7 位小数。
float64 是双精度浮点数类型，也称为 float，表示范围是约 -1.8e308 到 1.8e308，可以精确到大约 15 位小数。
